# Alphabet arabe — Site de démonstration

Ce dossier est prêt pour **GitHub Pages** (hébergement gratuit).

## Étapes
1. Crée un dépôt public sur GitHub (ex. `alphabet-isole`).
2. Envoie tous les fichiers de ce dossier à la racine du dépôt (`index.html`, dossier `audio/`).
3. Dans **Settings → Pages** :
   - Source : `Deploy from a branch`
   - Branch : `main` / folder `/ (root)`
4. Attends 1 à 2 minutes. L'URL publique sera :
   `https://TON-UTILISATEUR.github.io/alphabet-isole/`

### Tester
- Ouvre : `https://TON-UTILISATEUR.github.io/alphabet-isole/`
- Le bouton lecture doit fonctionner.
- Lien direct vers l'audio :
  `https://TON-UTILISATEUR.github.io/alphabet-isole/audio/alphabet_isole.wav`

### Intégration avec le Tome 1
Dans le fichier HTML du Tome 1, remplace l'URL provisoire du QR par l'URL publique réelle ci-dessus.
